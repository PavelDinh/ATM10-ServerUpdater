# ATM10-ServerUpdater
* Simple Minecraft modpack ATM10 server updater.
* Download and setup ATM10 server to run.
* Support backup for previous version.
* Shortcut for starting the latest server.

Running first time will just install ATM10 latest server version and run it.
If you have already running ATM10 server, the app will check for latest release and downloads it. 
It will also copy world folder with configs to the new version as well.
You can set it up in appsettings.json.

Keep in mind this app was intended for ATM10 server update purpose only, other modpacks may not work.
